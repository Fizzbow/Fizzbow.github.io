vue中动态添加类

point:

classList.add(‘active’) : 相当于在原有的类加上一个类,比如<div class = 'click active'></div>

当有一个需求需要实现点击的时候css可以实现相应的改变

  • data
    data() {
      return {
        activeClass: -1,// 0为默认选择第一个，-1为不选择
         list: [
          {title: '全网搜索(100w+)'},
          {title: '优惠推荐(包邮10w+)'},
        ],
      }
    },
  • html
 <ul class="list">
      <li :span="12" v-for="(v,k) in list" :key="k" :class="activeClass ==k?'active':''">
      <span @click="IsActive(k)">{{v.title}}</span>
      </li>
    </ul>
  • css
.active {
    border-bottom: 1px solid #FE4520;
    color: #FE4520;
  }
  • methods
    methods: {
      IsActive(k) {
        this.activeClass = k;
      }
    }

数组语法

<div :class="[activeClass, errorClass]"></div>
data() {
  return {
    activeClass: 'active',
    errorClass: 'text-danger'
  }
}
  • 渲染结果为：
    <div class="active text-danger"></div>

    2. v-enter-active：定义进入过渡生效时的状态
    3. v-enter-to：过渡结束状态

基于vue-router中的过渡效果

同样适用于上述class过渡
项目中使用

<transition>
  <router-view></router-view>
</transition>

const Parent = {
  data () {
    return {
      transitionName: 'slide-left'
    }
  },
  beforeRouteUpdate (to, from, next) {
    const toDepth = to.path.split('/').length
    const fromDepth = from.path.split('/').length
    this.transitionName = toDepth < fromDepth ? 'slide-right' : 'slide-left'
    next()
  },
  template: `
    <div class="parent">
      <h2>Parent</h2>
      <transition :name="transitionName">
        <router-view class="child-view"></router-view>
      </transition>
    </div>
  `
}
