git命令

2021-11-03

创建新分支

git checkout（切换分支） -b xx 创建新分支xx

git branch：查看当前分支

git status ：检查当前分支的文件状态

git add .：添加到暂存区

git push -u origin xx 第一次推送新分支xx（当github仓库没有xx分支时，要加上-u）

合并分支（合并xx到主分支）

git checkout main

git merge xx

git push 推送到云端
